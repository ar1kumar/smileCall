HOW TO USE

VIA MOBILE APPLICATIONS

  1. Enter the Brand Pin/Operator code of respective dialer downloaded.
  2. Enter the User ID and Password you received in your email against your plan purchase.
  3. Save the credentials and its ready to use.
  4. From the Dial screen, one can dial any country with country code following Number with Local/National area code with out any Zero prefix.
